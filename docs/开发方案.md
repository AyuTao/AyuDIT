# AyuDIT 基于 DaVinci Resolve 的工作流方案

## 项目目标
- 针对片场 DIT 流程，完成素材拷贝校验、自动导入至 DaVinci Resolve，并套用预设 LUT。
- 在 Resolve 内联动生成代理、同步音频、产出带缩略图的 PDF 报告。
- 以 Workflow Integration 插件为主界面，整合 Node.js/React 前端与 Python 脚本执行层。

## 系统架构概览
1. **Workflow Integration 插件 (Electron + React)**
   - 运行在 Resolve 内部窗口，负责 UI、任务管理与进度展示。
   - 通过安全的 IPC 通道调用预加载脚本，触发 Node 服务层。
2. **Node 服务层**
   - 调用 `WorkflowIntegration.node` 访问 Resolve JavaScript API。
   - 负责任务编排、状态持久化、与 Python 子进程通信。
3. **Python 任务引擎**
   - 执行素材拷贝校验（支持哈希比对）、代理生成调度、PDF 报告生成等重运算任务。
   - 通过 DaVinci Resolve Python API 调用（`DaVinciResolveScript`）。
4. **外部工具链**
   - ffmpeg：如需 Resolve 外生成代理或音频处理。
   - ReportLab/WeasyPrint：生成带缩略图和元数据的 PDF 报告。

## 关键功能流程
### 1. 素材拷贝与校验
- React 前端输入卡口路径与目标卷路径。
- Node 调用 Python 执行复制；使用 xxHash/MD5 进行完整性校验。
- 完成后记录任务状态（成功/失败/重试）。

### 2. 自动导入至 Resolve
- Python 通过 `MediaStorage.AddItemListToMediaPool()` 导入指定目录素材。
- 支持自动创建 Bin 结构（按拍摄日/机位）。
- 可选同步第三方元数据写入 `MediaPoolItem.SetMetadata()`。

### 3. 套用 LUT 与色彩模板
- 获取目标时间线或素材列表。
- 调用 `TimelineItem.SetLUT(nodeIndex, lutPath)`、`MediaPoolItem.SetClipProperty()` 自动套用预设。
- 支持配置 LUT 映射策略（按项目 / 摄影机型号 / 用户指定）。

### 4. 音频同步
- 针对导入素材执行 `MediaPool.AutoSyncAudio()`，支持波形或时码模式。
- 同步完成后更新任务状态并记录日志。

### 5. 代理生成（Resolve 内完成）
- 设置 `Project.SetRenderSettings({ ... })` 为代理编码模板（例如 ProRes Proxy）。
- 构建临时时间线或直接对媒体池素材执行生成：
  1. 创建代理 Render Preset。
  2. `Project.AddRenderJob()` 添加任务。
  3. `Project.StartRendering(jobIds)` 触发渲染并监控状态。
- 渲染完成后使用 `MediaPoolItem.LinkProxyMedia()` 挂载生成的代理文件。

### 6. PDF 报告生成
- 遍历导入素材：
  - `MediaPoolItem.GetMetadata()` 获取信息。
  - `Timeline.GetCurrentClipThumbnailImage()` 捕获缩略图（Base64）。
- Python 利用模板生成 PDF（含缩略图、文件信息、LUT、代理状态、音频同步结果）。

## 数据与状态管理
- 采用 SQLite 或轻量 JSON 记录任务实体：素材 ID、拷贝校验、导入、LUT、音频同步、代理、报告状态。
- 支持断点恢复与重复执行防护（根据媒体唯一 ID/哈希）。

## 安全与兼容性
- 遵循 Resolve v19 的 sandbox + contextIsolation 要求。
- 预加载脚本中仅暴露受控 API，Renderer 端不得直接访问 Node 模块或文件系统。
- 检查 Python 环境与外部依赖是否可用，不可用时在 UI 中提示安装指引。

## 实施计划
1. **原型阶段 (Week 1-2)**

   - 完成 Node ↔ Python 通道与基础任务管理模块。
2. **核心功能阶段 (Week 3-6)**

3. **报告与完善 (Week 7-8)**

   - UI 完善、任务日志、错误处理。
4. **测试与部署 (Week 9)**
   - 编写自动化脚本对关键流程进行回归测试。
   - 编写部署文档（插件安装、依赖安装）。

## 风险与对策
- **Render 队列阻塞**：代理渲染占用 Deliver 队列；方案是安排时段或在后台监控渲染进度并暂停普通任务。
- **权限与路径问题**：跨磁盘拷贝需处理 macOS 权限提示；提供预运行检测步骤。
- **外部依赖缺失**：ffmpeg、ReportLab 等检查后给出安装引导或集成捆绑版本。
- **Resolve API 变化**：持续关注 Blackmagic 文档更新，保持 `WorkflowIntegration.node` 最新版本。
